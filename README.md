施設使用申請システム (Official)
沖縄県体協スポーツ会館（公益財団法人沖縄県スポーツ協会）の施設使用申請書をデジタルで作成・計算・出力するためのシステムです。

概要
手書きの手間を省き、利用区分や時間帯に応じた複雑な料金計算を自動化します。入力した内容は、そのまま公的な申請書のフォーマット（A4サイズ）でPDF出力・印刷が可能です。

主な機能
自動料金計算:

「一般」「加盟団体」「高校生以下」の区分別料金に対応。

平日（昼間・夜間）および土日祝日の時間外割増を自動判定。

施設ごとの空調使用料の加算。

団体情報の自動入力: 加盟団体リストから選択することで、所在地や代表者名を自動補完。

マルチ予約対応: 最大3つの予約枠（異なる日時や施設）を同時に申請可能。

高精度な印刷レイアウト: 印刷時にはUI要素を隠し、公的な「（１号様式）施設使用申請書」の形式で出力。
